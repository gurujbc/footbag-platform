# Security Policy

This file covers vulnerability reporting only. For the project's security, privacy, and historical data governance policy, see [`docs/GOVERNANCE.md`](docs/GOVERNANCE.md).

Only the current `main` branch is supported.

## Reporting a vulnerability

**Do not use public GitHub issues for security reports.**

Use GitHub's [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability)
(Security tab → "Report a vulnerability"), or email **davidleberknight@gmail.com**
with subject `[SECURITY] footbag-platform — <description>`.

Please include: description and potential impact · steps to reproduce ·
affected components · suggested fix if known.

Acknowledgement as fast as possible, note this is a volunteer project. 
Coordinated disclosure once a fix is ready.
